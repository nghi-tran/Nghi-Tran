COMPLEX OBJECT

The first part of the project contains a physical drawing of a tall glass with a handle and pedalstal. This drawing is done by carefully observing a measuring the glass object. The arial and elevation view are first sketched out to used as reference for the three-dimensional view. Using the reference, the three-dimensional wireframe is built out of circles and ellipses.

The sketch version is then used as a guide on Adobe Illustrator to construct the wireframe digitally. The digital wireframe is built using careful measurements and calculations. This version consists of multiple ellipses and circles. This guide is then used to paint the light and dark versions of the glass.
